Detailed Description

The HTTP request.

Contains request method and the URL along with the HTTP version, list of HTTP headers and request body content derived from Message base class.

This class is designed to be used as dynamic object created with new operator. There are a lot of factory methods:

  • create() creates custom request
  • GET() creates GET request
  • PUT() creates PUT request
  • POST() creates POST request

The request HTTP method and the URL cannot be changed after creation.
