Detailed Description

The REST Server submodule.

This is helper class.

You have to call initRestServerModule() method before use any of the class methods. The best place to do that is static factory method of your application.

Examples:
examples/simple_dev.hpp, examples/simple_gw.hpp, and examples/zigbee_gw.hpp.
