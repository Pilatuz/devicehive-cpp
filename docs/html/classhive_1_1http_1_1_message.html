Detailed Description

The HTTP message.

This is base class for HTTP requests and responses. Contains HTTP version, list of HTTP headers, and the body content. The HTTP version may be changed using setVersion() method. The body content stored as string using setContent() method.

All headers are stored in map:

  • headers are case insensitive
  • headers are sorted by name
