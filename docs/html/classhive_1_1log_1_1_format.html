Detailed Description

The log message format.

This is base class for all log message formats. Derived classes should override the format() virtual method.

The create() factory method should be used to get a new instance of the Format class.

By default simple format is used:

timestamp logger level message

This format is always available through defaultFormat() static method.

Also there is possible to use custom format (with printf-like syntax) which is available through customFormat() static method. For example, default format is equivalent to "%T %N %L %M\n". The following format options are allowed:

option description
%T the timestamp
%N the logger name
%L the logging level
%M the log message (with prefix)

The auxiliary getLevelName() static method might be used to convert logging level into the text representation.
