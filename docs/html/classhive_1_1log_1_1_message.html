Detailed Description

The log message.

This class holds all information about log message.

You have to deal with this class directly only if you develop your own log target or formatter. See hive::log::Target::send() and hive::log::Format::format() methods.

Warning
This class doesn't make any copies of strings. Be careful with strings lifetime.
